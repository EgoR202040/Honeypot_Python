# Honeypot для обнаружения сетевых сканирований

## Описание

Python honeypot для обнаружения и анализа сетевых сканирований и атак. Эмулирует различные сервисы (SSH, HTTP, HTTPS) и отслеживает подозрительную активность.

## Функциональность

- Эмуляция SSH, HTTP, HTTPS серверов
- Обнаружение различных типов сканирования:
  - TCP SYN сканирование
  - TCP Connect сканирование
  - UDP сканирование
  - Агрессивное сканирование
  - Скрытное сканирование
- Расширенное логирование всех соединений
- Статистика и анализ активности
- Генерация отчетов и предупреждений
## Запуск
python honeypot.py --ports [номера портов] \n Для получения справки используйте параметр -h или --help
